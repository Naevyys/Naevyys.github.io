Learning Multimodal Information: Definition and Experimentation

Bachelor Semester Project 6 - Summer semester 2020/2021

Project Academic Tutor: Prof. Dr. Nicolas Guelfi

Project description: Modern AI systems become increasingly powerful tools, but there remain many tasks that humans perform much better. We suppose that the multimodal perception and interpretation of the world through our five senses significantly contributes to this superior performance of humans. Involving multiple modalities in the decision making process of an AI system would therefore have the potential of improving the accuracy of its predictions, and the research field of multimodal machine learning is focussing on this problem. This semester project aims to introduce the student to this research field.

Main technologies: Python

Final report

SmartSparks - A Crowdsourcing App for Organisations

Bachelor Semester Project 5 - Winter semester 2020/2021

Project Academic Tutor: Prof. Dr. Steffen Rothkugel

Project description: Crowdsourcing is a powerful and sometimes underestimated technique to gather data from the crowd. It can be used for instance in scientific research to gather necessary scientific data through or from the entire population, or within companies or groups of people to gather new ideas more efficiently. However, in the latter case, involving a large group of people in the common process of innovation is difficult for reasons such as lack of communication means between people and a large amount of ideas out of which the bests need to be selected. The aim of this project is to create a prototypical app which will help companies to perform crowdsourcing among their employees and eventually also their customers.

Main technologies: Flutter, Dart, Firebase

Final report

Explore Explainable AI with QT

Bachelor Semester Project 3 - Winter semester 2019/2020

Project Academic Tutor: Amro Najjar (and Diego Agustin Ambrossio)

Project description: Artificial intelligence is a wide and complex field of computer science, closely related to robotics. During this project, the student will program the university robot QT to play music according to the facial expression of the human in front of her, i.e. if the human looks sad, QT will play sad music, and if the human looks happy, the music should be happy too. Through this practice, the student will explore and work on the domains of Human-Robot Interaction and Explainable AI, which will allow her to describe and explain the QTâ€™s behaviour with the human.

Main technologies: Python, ROS, OpenCV, QTrobot

Final report

ObDrop - A Mobile Application using Augmented Reality

Bachelor Semester Project 2 - Summer semester 2018/2019

Project Academic Tutor: Dr. Jean Botev

Project description: Sharing georeferenced information between users is common to many mobile applications and services; far less common is the location-based sharing of virtual objects. In this project, the student will develop a prototypical, AR-based mobile application that enables users to interactively augment the real world with such objects, which are then shared with and visible to the other users.

Main technologies: Swift, ARKit, SQLite

Final report

A Web-based Editing Interface for Georeferenced Data

Bachelor Semester Project 1 - Winter semester 2018/2019

I won the first place of the BiCS Student Project Award with this BSP. Find out more here.

Project Academic Tutor: Dr. Jean Botev

Project description: Georeferenced data is underlying many of the most exciting computer systems and services today. In this project, the student will explore how such data can be created and managed, using the latest APIs and standards for creating interactive, web-based applications such as HTML5. Aside from a taxonomy of the available technologies and platforms, the project shall result in a prototypical design and implementation for such a management interface.

Main technologies: HTML, CSS, JavaScript, Google Maps API
